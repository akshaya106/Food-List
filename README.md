Food List Analysis Project
Project Overview
This project analyzes a food dataset to understand food popularity, customer preferences, ratings, and order trends. The analysis is performed using Python, Pandas, and Matplotlib.
Objectives
Analyze food data.
Identify popular food items.
Calculate average ratings.
Find total orders and spending.
Visualize food trends using charts.
Dataset Information
The dataset contains:
Food Name
Customer Name
Customer ID
City
Total Orders
Total Spending
Rating
